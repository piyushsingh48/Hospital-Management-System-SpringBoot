# Hospital Management System – Spring Boot Application

## 📌 Project Overview

Hospital Management System is a web-based application developed using Spring Boot that helps manage hospital operations such as department, patient records, 
doctor information, appointments, and administrative tasks. The system improves efficiency by digitizing hospital management processes.

## 🚀 Features

* Patient Registration and Management
* Doctor Management
* Appointment Booking System
* View Patient Records
* Department Management
* Prescription of Medical Record

## 🛠️ Technologies Used

* Spring Core
* Spring Boot
* Spring MVC
* Spring Data JPA
* PostgreSQL
* Postman
* Maven
* ApacheTomcat

## 📂 Project Structure

src/main/java

* controller
* service
* repository
* entity
* configuration

src/main/resources

* templates
* static
* application.properties

## ⚙️ Setup Instructions

1. Clone the repository
   git clone https://github.com/piyushsingh48/Hospital-Management-System-SpringBoot.git

2. Import the project into Eclipse 

3. Configure the MySQL database in **application.properties**

4. Run the Spring Boot application

5. Open the browser and access the application at
   http://localhost:8080
